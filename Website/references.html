Texto Crítico

  • : hay versos extras en los testimonios indicados.
  • ≠: los testimonios indicados tienen variaciones muy importantes en este verso. En muchos casos, el verso ha sido reemplazado por uno completamente diferente, que simplemente posee el mismo lugar dentro del texto.
  • : el verso se encuentra en diferentes ubicaciones en los distintos testimonios.
  • : el verso se encuentra ausente en los testimonios indicados.
  • : nota crítica.
  • finaliza el testimonio indicado.

Aparato

  • add. = addit = “agrega”, se utiliza para los versos extras que no están en el texto crítico. No implica un juicio de valor sobre la autenticidad de los mismos.
  • ante = "antes" = indica la palabra antes de la cual un testimonio contiene texto extra.
  • cett. = ceteri codices = “los otros manuscritos”. Usado para indicar que una lectura se encuentra en todos los otros manuscritos, excepto el manuscrito guía
  • del. = delevit = “borrado”, significa que el copista eliminó texto, por medio de tachaduras, subrayado o borrado.
  • exp. = explicit = "finaliza"; indica el final de un testimonio.
  • h. v. om. = hic versus omissum= “este verso está ausente”.
  • inc. = incipit = "comienza"; indica el comienzo de un testimonio.
  • lemma h.v. = lemma huius versus = "lemma de este verso"; cuando se utiiza un testimonio diferente al manuscrito guía para construir el texto crítico de un verso.
  • om. = omissum= “ausente”.
  • transp. = transposuit = “transpuesto”, se utiliza para indicar que el orden de los versos ha sido alterado (si bien también se ha utilizado en otras ediciones a nivel de palabras, lo reservo para transposiciones a nivel de líneas).

Critical Text

  • : extra verses.
  • ≠:important variations in this verse. It might have been replaced by a different one.
  • : the verse is in transposed.
  • : absent verse.
  • : critical note.
  • end of witness.

Critical Apparatus

  • add. = addit = “adds”.
  • ante = "before". Word before which something was added.
  • cett. = ceteri codices = “the other witnesses”. Used to say a reading is in all witnesses except for the guide manuscript.
  • del. = delevit = “deleted”, the writer signals the deletion of some text, either with an overstrike, underline or erasure.
  • exp. = explicit = "ends"; indicates the ending of a witness.
  • h.v.om. = hic versus omissum= “this verse is absent”.
  • inc. = incipit = "begins"; beginning of a fragmentary witness.
  • lemma h.v. = lemma huius versus = "lemma of this verse"; the witness used to construct the critical text when the guide manuscript is not.
  • om. = omissum= “absent”.
  • transp. = transposuit = “transposed”.
